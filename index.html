Artwork Ontology

This page provides information about the Artwork ontology resources.

Classes

  • Artwork: Represents an artwork.
  • Artist: Represents an artist.
  • ArtworkDimension: Represents the dimensions of an artwork.

Datatype Properties

  • hasId: Datatype property for the ID of an artwork.
  • hasAccessionNumber: Datatype property for the accession number of an artwork.
  • hasArtistRole: Datatype property specifying the role of an artist related to an artwork.
  • hasTitle: Datatype property for the title of an artwork.
  • hasCreditLine: Datatype property for the credit line of an artwork.
  • madeInYear: Datatype property indicating the year an artwork was made.
  • acquiredInYear: Datatype property indicating the year an artwork was acquired.
  • hasInscription: Datatype property for any inscription on an artwork.
  • hasThumbnailCopyright: Datatype property for copyright information related to the thumbnail of an artwork.
  • hasThumbnailUrl: Datatype property for the URL of the thumbnail of an artwork.
  • hasUrl: Datatype property for the URL of an artwork.

Object Properties

  • hasArtist: Object property linking an artwork to its artist.
  • hasMedium: Object property specifying the medium of an artwork.
  • hasDimension: Object property specifying the dimensions of an artwork.
