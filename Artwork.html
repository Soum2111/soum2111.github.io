Artwork Ontology

This page provides information about the Artwork ontology resources.

Classes

  • Artwork: Represents an artwork.
  • Artist: Represents an artist.
  • ArtworkDimension: Represents the dimensions of an artwork.

Datatype Properties

  • hasId: Datatype property for the ID of an artwork.
  • hasAccessionNumber: Datatype property for the accession number of an artwork.

Object Properties

  • hasArtist: Object property linking an artwork to its artist.
  • hasMedium: Object property specifying the medium of an artwork.
