Artwork Ontology

This page provides information about the Artwork ontology resources.

Classes

  • Artwork: Represents an artwork.
  • Artist: Represents an artist.
  • ArtworkDimension: Represents the dimensions of an artwork.

Datatype Properties

  • hasId: Datatype property for the ID of an artwork. URL: https://soum2111.github.io/hasId
  • hasAccessionNumber: Datatype property for the accession number of an artwork. URL: https://soum2111.github.io/hasId
  • hasArtistRole: Datatype property specifying the role of an artist related to an artwork. URL: https://soum2111.github.io/hasArtistRole
  • hasTitle: Datatype property for the title of an artwork. URL: https://soum2111.github.io/hasTitle
  • hasCreditLine: Datatype property for the credit line of an artwork. URL: https://soum2111.github.io/hasCreditLine
  • madeInYear: Datatype property indicating the year an artwork was made. URL: https://soum2111.github.io/madeInYear
  • acquiredInYear: Datatype property indicating the year an artwork was acquired. URL: https://soum2111.github.io/acquiredInYear
  • hasInscription: Datatype property for any inscription on an artwork. URL: https://soum2111.github.io/hasInscription
  • hasThumbnailCopyright: Datatype property for copyright information related to the thumbnail of an artwork. URL: https://soum2111.github.io/hasThumbnailCopyright
  • hasThumbnailUrl: Datatype property for the URL of the thumbnail of an artwork. URL: https://soum2111.github.io/hasId
  • hasUrl: Datatype property for the URL of an artwork. URL: https://soum2111.github.io/hasUrl
  • hasArtist: Object property linking an artwork to its artist. URL: https://soum2111.github.io/hasArtist
  • hasMedium: Object property specifying the medium of an artwork. URL: https://soum2111.github.io/hasMedium
  • hasDimension: Object property specifying the dimensions of an artwork. URL: https://soum2111.github.io/hasDimension
